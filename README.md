# Hospital Readmission Prediction

## Overview

Explore the intricacies of predicting hospital readmission using machine learning algorithms. This project analyzes patient data, admission details, and medical history to develop a robust model for predicting readmission risks.

## Key Features

- **Demographics:**
  - Age and Gender analysis.

- **Admission Details:**
  - Patterns related to Admission Type and Emergency Visits.

- **Medical History:**
  - Impact of lab procedures, medications, and diagnoses.

- **A1C Test Results:**
  - Influence on the likelihood of readmission.

## Approach

1. **Data Exploration:**
   - Uncover insights from patient demographics, admission types, and medical history.

2. **Feature Engineering:**
   - Transform data for improved model performance.

3. **Model Development:**
   - Employ diverse classification algorithms for predicting readmission risks.

4. **Model Evaluation:**
   - Analyze model performance metrics to identify the most effective algorithm.

## Best Performing Model

- **K-Nearest Neighbors (KNN):**
  - Achieved high accuracy in predicting hospital readmission.

## Insights and Discoveries

- **Influential Factors:**
  - Identified key factors affecting readmission risks.

- **KNN Superiority:**
  - KNN demonstrated exceptional predictive capabilities.
